To-Do List Application

Консольное приложение для управления персональными задачами с поддержкой долгосрочного хранения данных. Проект реализует базовые CRUD-операции (Create, Read, Delete) и демонстрирует навыки работы с файловой системой.
Функциональные возможности

    Персистентность данных: Автоматическое сохранение и загрузка задач из локального файла tasks.txt.

    Управление списком: Потоковое добавление и удаление задач с мгновенной синхронизацией данных.

    Валидация индексов: Безопасное удаление элементов по порядковому номеру с обработкой исключений ValueError.

    Автоматическая нумерация: Динамическое формирование списка с использованием индексации в реальном времени.

Технические особенности
Работа с файловой системой (File I/O)

Приложение использует менеджер контекста with и кодировку UTF-8 для обеспечения корректной работы с кириллицей. Проверка существования файла реализована через модуль os, что предотвращает ошибки при первом запуске программы.
Архитектура данных

Задачи хранятся в оперативной памяти в виде динамического списка (list). Синхронизация с жестким диском происходит после каждого изменения состояния (добавление/удаление), что минимизирует риск потери данных при аварийном завершении.
Структура проекта

    to-do-list.py — основной исполняемый файл.

    tasks.txt — текстовая база данных проекта.

    .gitignore — файл конфигурации для исключения временных данных из репозитория.

    README.md — техническая документация.

Установка и запуск

    Bash
