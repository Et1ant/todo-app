To-Do List Application

Консольное приложение для управления персональными задачами с поддержкой долгосрочного хранения данных. Проект реализует базовые операции со списками и демонстрирует навыки работы с файловой системой и обработки пользовательского ввода.
Функциональные возможности

    Персистентность данных: Автоматическое сохранение задач в файл tasks.txt и их загрузка при старте приложения.

    Управление задачами: Добавление новых пунктов и удаление существующих по порядковому номеру.

    Валидация ввода: Защита от ошибок при вводе некорректных номеров задач или нечисловых значений с помощью блоков try-except.

    Автоматическая нумерация: Использование функции enumerate для динамического формирования списка задач в интерфейсе.

Технические особенности
Работа с файловой системой

Для обеспечения сохранности данных используется менеджер контекста with open() с кодировкой utf-8. Программа проверяет наличие файла базы данных через os.path.exists, что гарантирует стабильную работу при первом запуске без предварительно созданных файлов.
Синхронизация состояния

Приложение работает по принципу немедленной синхронизации: каждое изменение списка задач в оперативной памяти (добавление или удаление) вызывает автоматическую перезапись файла tasks.txt. Это предотвращает потерю данных при внезапном закрытии терминала.
Структура проекта

    to-do-list.py — основной файл с логикой приложения.

    tasks.txt — текстовый файл для хранения списка задач.

    .gitignore — файл конфигурации Git для исключения временных данных из репозитория.

    README.md — техническая документация.
